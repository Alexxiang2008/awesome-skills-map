# 访谈写作配置

> 此文件用于存储你的写作偏好，首次使用时会自动生成

---

## 字数设置

- **目标字数**：1000
- 可选范围：500 / 800 / 1000 / 1500 / 2000

| 字数 | 适用场景 |
|------|----------|
| 500 | 短平快，朋友圈/微博 |
| 800 | 朋友圈适配，轻阅读 |
| 1000 | 公众号标准（推荐） |
| 1500 | 深度故事，长文阅读 |
| 2000 | 长篇特稿，深度报道 |

---

## 风格设置

- **当前风格**：故事型

### 可选风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **故事型** | 场景切入、情绪细腻、金句点睛 | 公众号、朋友圈 |
| **干货型** | 观点鲜明、逻辑清晰、案例支撑 | 知乎、行业媒体 |
| **对话型** | 保留问答结构、口语化、真实感强 | 播客文稿、社群分享 |

---

## 金句密度

- **当前设置**：每300字符1个金句
- 可选范围：每200字符 / 每300字符 / 每500字符

| 密度 | 效果 |
|------|------|
| 每200字符 | 高密度，适合短文、金句集 |
| 每300字符 | 标准密度（推荐） |
| 每500字符 | 低密度，适合叙事为主的长文 |

---

## 标题风格偏好

勾选你偏好的标题类型（可多选）：

- [x] 悬念型：制造信息差，让人想点进来
- [x] 情绪型：引发共鸣，让人觉得"这说的是我"
- [x] 反常识型：挑战认知，让人觉得"这和我想的不一样"

---

## 使用说明

1. 修改上方配置后保存文件
2. 下次使用 `/访谈` 时会自动读取此配置
3. 如需临时覆盖，可在命令中指定，如：`/访谈 -字数=1500 -风格=干货型`
