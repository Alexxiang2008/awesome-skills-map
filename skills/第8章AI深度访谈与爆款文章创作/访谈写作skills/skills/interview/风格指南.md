# 三种写作风格指南

> 本文档定义了故事型、干货型、对话型三种写作风格的详细标准

---

## 风格对比总览

| 维度 | 故事型 | 干货型 | 对话型 |
|------|--------|--------|--------|
| **开头** | 场景切入，制造悬念 | 抛出核心观点或问题 | 保留第一个提问 |
| **结构** | 起承转合，情绪曲线 | 总分总，逻辑递进 | 问答交替，自然流动 |
| **语言** | 文学化，有画面感 | 简洁直接，重信息量 | 口语化，保留语气词 |
| **金句** | 情绪共鸣型 | 认知颠覆型 | 真实感悟型 |
| **适合场景** | 公众号、朋友圈 | 知乎、行业媒体 | 播客文稿、社群分享 |

---

## 故事型风格

### 核心特征

- **场景先行**：用具体的时间、地点、画面开场
- **情绪细腻**：捕捉微妙的心理变化，不用"很难""很开心"这类笼统词
- **金句点睛**：在情绪高点或转折处放置金句，强化记忆点

### 结构模板

```
【开头】场景切入 → 制造悬念，让人想知道"然后呢"
    ↓
【背景铺垫】快速交代必要信息 → 不超过2段
    ↓
【冲突展开】核心矛盾 + 情绪颗粒度 → 故事主体
    ↓
【至暗时刻】最难的那一刻 → 情绪高潮
    ↓
【转折/突破】怎么挺过来的 → 给人希望
    ↓
【升华/金句】提炼洞察 → 让人想收藏
    ↓
【结尾】余韵/开放性 → 意犹未尽
```

### 开头示例

```markdown
2019年11月的一个下午，张磊在会议室里听到了那句话。

"我不干了。"

合伙人把工牌放在桌上，头也不回地走了。那一刻，张磊脑子里只有一个念头：
下个月的工资，发不出来了。
```

### 金句类型

- **情绪共鸣型**："真正的勇敢不是不怕，而是怕得要死还往前走"
- **画面感型**："焦虑就像凌晨三点的闹钟，你知道该睡，但就是睡不着"
- **反转型**："那天我才明白，放弃才是最难的选择"

## 干货型风格

### 核心特征

- **观点先行**：开篇直接抛出核心观点或问题
- **逻辑清晰**：总分总结构，每段有明确的论点
- **案例支撑**：用具体案例和数据证明观点

### 结构模板

```
【开头】核心观点/问题 → 直接告诉读者"这篇文章讲什么"
    ↓
【论点1】第一个分论点 + 案例/数据支撑
    ↓
【论点2】第二个分论点 + 案例/数据支撑
    ↓
【论点3】第三个分论点 + 案例/数据支撑
    ↓
【总结】回扣核心观点 + 行动建议
```

### 开头示例

```markdown
创业最大的风险不是没钱，而是合伙人在关键时刻掉链子。

张磊的故事就是一个典型案例。融资前一周，合伙人突然撂挑子，
直接导致项目停摆三个月，烧掉了80万现金流。

这篇文章，我想聊聊如何避免这种"合伙人风险"。
```

### 金句类型

- **认知颠覆型**："效率是把事情做对，效能是做对的事情"
- **数据冲击型**："10个外包项目，能上线运营的不到3个"
- **方法论型**："选合伙人，看的不是能力，是退出成本"

---

## 对话型风格

### 核心特征

- **保留问答**：保持访谈的问答结构
- **口语化**：保留语气词、停顿、口头禅
- **真实感强**：不过度润色，保持原汁原味

### 结构模板

```
【开头】第一个提问 → 直接进入对话
    ↓
【问答1】关键问题 + 真实回答
    ↓
【问答2】追问细节 + 情绪表达
    ↓
【问答3】转折/高潮 + 深度回答
    ↓
【问答4】总结/感悟 + 金句
    ↓
【结尾】开放性问题或感悟
```

### 开头示例

```markdown
**Q：创业失败这件事，你是在哪个瞬间意识到"完了"的？**

A：合伙人在融资前一周突然说不干了。那一刻我脑子一片空白。

**Q：他说这话的时候，你们是在哪里？**

A：在会议室。他把工牌往桌上一放，说"我不干了"，然后就走了。
我愣在那儿，大概有十秒钟没反应过来。
```

### 金句类型

- **真实感悟型**："那时候我才明白，创业不是打怪升级，是活着就是胜利"
- **口语化型**："说实话，当时真的想过放弃，但想想欠的钱，又不敢"
- **反思型**："现在回头看，那个决定其实是对的，只是当时不知道"

## 风格选择建议

| 场景 | 推荐风格 |
|------|----------|
| 公众号日常发文 | 故事型 |
| 朋友圈分享 | 故事型（短版） |
| 知乎回答 | 干货型 |
| 行业媒体投稿 | 干货型 |
| 播客文稿 | 对话型 |
| 社群分享 | 对话型 |
| 品牌故事 | 故事型 |
| 方法论输出 | 干货型 |
