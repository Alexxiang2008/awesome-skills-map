# 发布到微信公众号

将 Markdown 文章一键排版并发布到微信公众号草稿箱。

## 参数

$ARGUMENTS - 要发布的 Markdown 文件路径（可选；如果不提供，请询问用户）

## 执行步骤

请严格按照 `skills/wechat-publish/SKILL.md` 中定义的流程执行：

1. 检查 `~/.wechat-publish/config.json` 是否已初始化；如未初始化，按 Skill 引导用户配置微信发布 API Key、图床和封面图 API。
2. 读取用户指定的 Markdown 文件，确认文章标题、摘要、作者、封面需求和主题风格。
3. 如用户需要封面图，使用 Skill 中的封面生成流程生成或选择封面。
4. 将文章内图片上传到配置的图床，并替换为可被微信公众号访问的图片链接。
5. 按所选主题把 Markdown 转换为微信公众号兼容 HTML。
6. 调用已配置的微信发布服务创建草稿或发布。

## 安全要求

所有密钥、Token、图床配置和发布服务地址必须从 `~/.wechat-publish/config.json` 或环境变量读取，不要在命令文件中硬编码任何凭据。
